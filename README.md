
# Student Expenses Management

Student Expenses Management is a Java application built to help students track and manage their daily or monthly expenses in a simple and organized way.

## Features
- Record student expenses
- Organize spending information
- Help users monitor personal budgets
- Improve financial awareness for students

## Tech Stack
- Java
- OOP principles
- File handling / data management

## Purpose
I built this project to solve a practical student problem by creating a simple system that can help manage spending and encourage better budgeting habits.
